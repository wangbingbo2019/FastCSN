# Results

This folder stores generated outputs from the original local FastCSN analyses.

- `benchmarks/`: CSV tables from accuracy, scalability, and sensitivity experiments.
- `asd_analysis/`: ASD vs Control differential network outputs, including average networks, differential edges, DE results, DNG lists, and generated ASD figures.
- `ad_vs_control/`: inferred condition table from the AD vs Control FastCSN run.

The large raw datasets needed to regenerate these outputs are documented in `../data/README.md` and are intentionally not committed.
